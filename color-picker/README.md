This is a color picker nodejs server, which is used to edit the mod config colors for developers.

usage:
```shell
cd color-picker
node index.js

# open web browser http://localhost:8000, and press F12.
# edit the background color

# click the button on the website, and the cpp file will be changed
```

--------

the `cd color-picker` is not required, only if your `const modConfigFile = "..\\include\\ModConfig.hpp"` have right relative path.
